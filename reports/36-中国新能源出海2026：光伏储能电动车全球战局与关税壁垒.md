# 36 — 中国新能源出海2026：光伏储能电动车全球战局与关税壁垒

**研究日期：** 2026年3月29日  
**课题方向：** 全球供应链 · 新能源 · 贸易壁垒

---

## 一、核心结论

2026年，中国新能源出海正从"产品出海"升级为"生态出海"，但面临美欧双重壁垒绞杀。美国关税最高达**145%**（叠加301条款），欧盟则祭出《工业加速器法案》（IAA）和《新电池法》的"规则高墙"。**H1洞察：真正的机会在中东、东南亚、南美等新兴市场——2026年2月，中国企业单月斩获海外储能订单**35.71GWh**，这些市场的贸易壁垒相对较低。同时，赴欧建厂（宁德时代匈牙利/西班牙、比亚迪波兰）和本地化运营是穿透欧美壁垒的唯一路径。**

---

## 二、"新三样"出海规模

### 2.1 数据总览

2023年，"新三样"（新能源汽车、锂电池、光伏）出口首次突破**万亿元**大关。2025年这一趋势加速：

- 锂电池出口同比增长**26.2%**
- 风力发电机组出口同比增长**48.7%**
- 电动摩托车及脚踏车出口同比增长**18.1%**
- 铁道电力机车出口同比增长**27.1%**

### 2.2 储能出海爆发

**2026年2月单月数据：**
- 中国企业海外斩获**30个储能订单**，总计**35.71GWh**
- 呈现"规模放大、主体多元、区域广泛"三大特征
- 得益于国内退税政策调整和美国关税阶段性调整

**广东储能出海数据（全国风向标）：**
- 2025年1-8月，广东省锂电池出口**126.61亿美元**，同比增长**39.7%**
- 广东出口德国、美国合计占比**35.7%**，广东出口额位居全国首位
- 德国首次超越美国成为中国锂电池**第一大出口市场**，对德出口额达**65.2亿美元**，海外占比19.1%

## 三、美国壁垒：关税大棒与《One Big Beautiful Bill Act》

### 3.1 关税体系

| 产品 | 关税水平 | 备注 |
|------|---------|------|
| 中国电动车 | **100%（301关税）** + 2.5%基础关税 = **102.5%** | 2024年5月从25%提升 |
| 中国锂电池/储能 | **145%**（部分品类叠加） | 综合税率约40.9%（2025年5月中美日内瓦共识后调整）|
| 中国光伏组件 | 取消柬埔寨/马来西亚/泰国/越南豁免，重启双反 | |
| 石墨等关键原材料 | 反倾销措施叠加 | |

### 3.2 《One Big Beautiful Bill Act》（OBBBA）

2026年，美国将《通胀削减法案》（IRA）中的"外国实体关注名单"（FEOC）扩大为**"受禁止的外国实体"（PFE）**，并设定了逐年递增的"非PFE比例"要求：

| 年份 | 电池组件最低本土比例（45X抵免） |
|------|-----------------------------|
| 2026年 | 60% |
| 2027年 | 65% |
| 2028年 | 70% |
| 2029年 | 80% |
| 2030年+ | 85% |

**目的：** 通过税收抵免排除规则，削弱中国企业在美国的竞争力。

### 3.3 关税对市场的影响

Wood Mackenzie分析：
- 在"贸易紧张"情景下（2026年底有效税率10%，对华34%），美国太阳能项目成本比欧洲高**54%**，比中国高**85%**
- 在"贸易战"情景下（2030年前激进关税），太阳能项目建造成本增加约**15%**
- 美国电池进口已降至**5年来最低水平**（Section 301关税提至25%后）

### 3.4 "绕道"模式的可持续性

中国企业的应对策略：通过东南亚、墨西哥、匈牙利等地设厂转口。但：
- 美国将审查《美墨加协定》原产地规则，可能**进一步提高本土比例要求**（现有65-75%）
- OBBBA法案将"实际控制"概念扩大，即便技术授权模式也可能被认定为中国控制
- 宁德时代-福特模式（技术授权+许可）在建项目是否豁免**尚无定论**

## 四、欧盟壁垒：规则高墙与IAA法案

### 4.1 欧盟IAA法案（Industrial Accelerator Act）

欧盟委员会已于**2026年3月4日**将IAA法案提交议会审议，核心目标：
- 将制造业占GDP比例从14.3%提升至**20%**（2035年）
- 对"特定国家产能占比高"的外国投资者施加**强制性技术转让、员工本地化、外资持股比例限制**

**覆盖领域：** 钢铁/水泥/铝业等基础原材料、光伏/风电/电池/热泵/氢能/碳捕获等净零技术、汽车零部件制造。

### 4.2 《新电池法》与碳足迹要求

**全生命周期合规挑战：**
- 碳足迹核算：要求电池产品全生命周期碳排放达标
- 材料回收比例：强制要求回收比例
- 供应链尽职调查：穿透式管理上游供应商环境与社会责任
- **电池护照**：数字化溯源，2025年已生效

### 4.3 碳边境调节机制（CBAM）

- 英国宣布自2027年起实施UK CBAM
- 澳大利亚、日本、韩国等亦在计划建立类似机制
- 2025年9月，欧洲太阳能制造委员会（ESMC）请求将光伏下游产品纳入CBAM

**全球绿色壁垒蔓延，中国储能企业面临更复杂外部环境。**

### 4.4 欧洲储能市场：规模扩张，格局重构

**2025年欧洲储能数据：**
- 电池储能安装量达**27.1-29 GWh**，同比增长**36-45%**
- 欧洲大储正从户储拉动转向大储拉动（电网灵活调节需求）
- 欧洲2023-2024年储能市场规模分别达23GWh（同比+156%）

**中国企业在欧洲的市场份额变化：**
- 阳光电源在欧洲市场份额从2023年的10%跃升至2024年的**21%**（同比增长67%）
- Tesla仍为全球最大BESS生产商，但与阳光电源差距在缩小

## 五、三大赛道出海现状

### 5.1 光伏：从关税战到碳足迹围剿

**关键数据：**
- 2019-2022年，中国对欧洲光伏出口从342亿元增至**1562亿元**（占当年总出口50.7%）
- 2023-2024年欧洲光伏增速放缓，但中国仍是全球最大供应国
- 全球光伏产能结构性过剩，价格持续承压

**壁垒演进：** 从"双反关税"→"碳足迹围剿"，合规成本急剧上升

### 5.2 储能：中国企业的最强出海赛道

**2025年上半年数据：**
- 中国储能电池出货量达**232.03GWh**，同比增长**118.4%**
- 中东、东南亚等新兴市场快速起量，大储市场加速增长
- 比亚迪与Grenergy达成智利阿塔卡马绿洲项目第六期供货，供应**3.5GWh**，总合作规模提升至6.5GWh

**典型出海项目：**

| 企业 | 地点 | 项目 |
|------|------|------|
| 宁德时代 | 匈牙利德布勒森 | 已投产运营 |
| 宁德时代 | 西班牙（与Stellantis合资） | 建设中 |
| 阳光电源 | 欧洲多国 | 市场份额21% |
| 比亚迪 | 波兰Greenvolt项目 | 1.6 GWh |
| 海辰储能 | 东欧 | 2 GWh长时储能协议 |
| 中国能建+广东院 | 菲律宾 | 1.4GW光伏+3.3GWh储能 |
| 博钠新能源 | 哈尔滨（国内，但钠离子技术输出） | 2GWh钠离子电池 |

### 5.3 新能源汽车：从"卖车"到"建生态"

**出海2.0阶段：** 整车贸易为主→本地建厂+生态建设

**主要市场壁垒：**
- 欧盟：已完成"调查—登记—临时税—正式税—追溯征税"全流程
- 美国：102.5%综合税率，基本封死出口通道

**突破路径：**
- 在东南亚（泰国）、欧洲（匈牙利、西班牙）、拉美（巴西）建厂
- 奇瑞新能源西班牙重启工厂，极星汽车南卡罗来纳州扩产
- 比亚迪电动汽车已出口至东南亚、澳大利亚、欧洲（匈牙利工厂）

## 六、出海战略：从"被动应对"到"主动合规"

### 6.1 多元市场分散风险

**新兴市场优先级（壁垒低+高增长）：**
- **中东**： Saudi Arabia、阿联酋、卡塔尔——能源转型需求旺盛，一带一路重点区域
- **东南亚**： 菲律宾、越南、泰国——电力缺口大，大储需求旺盛
- **南美**： 巴西、智利——锂资源+储能需求双驱动
- **非洲**： 电力基础设施薄弱，可再生能源+储能空间大

**"一带一路"上半年清洁能源投资已超去年全年**，中东大单频出（比亚迪15.1GWh沙特项目）。

### 6.2 本地化建厂策略

| 目的地区域 | 建厂策略 | 代表案例 |
|-----------|---------|---------|
| 欧洲 | 穿透欧盟壁垒 + 进入欧洲供应链 | 宁德时代（匈牙利+西班牙）、隆基绿能（德国） |
| 北美 | 规避高关税（部分） | 宁德时代-福特技术授权模式 |
| 东南亚 | 规避美国关税 + 区域供应链 | 亿纬锂能东南亚工厂 |
| 墨西哥 | 近岸外包 | 辐射北美市场 |

### 6.3 合规能力建设

**数字化+绿色化转型：**
- 区块链+大数据实现从矿山到回收的全生命周期溯源
- 满足欧盟电池法"电池护照"要求
- 建立碳足迹核算体系，对接国际LCA标准

**钠离子电池：供应链安全冗余**
- 宁德时代第二代钠电池成本已接近磷酸铁锂电池
- 博钠新能源哈尔滨项目（2GWh钠离子，2025年底投产）
- 减少对锂资源的依赖，降低供应链地缘政治风险

### 6.4 标准制定参与

- 联合东南亚、中东国家推动区域性储能标准制定
- 以"技术换资源"模式与资源国建立合作（减少锂/镍/钴采购风险）
- 利用WTO争端解决机制应对贸易调查

## 七、总结与投资方向

### 7.1 核心结论

中国新能源出海已从"成本竞争"升级为"体系竞争"——产品力只是入场券，合规能力+本地化运营+供应链韧性才是决胜关键。

**三重壁垒的本质：**
- 美国"关税大棒"：用高成本直接压制进口
- 欧盟"规则高墙"：用合规体系构建长期壁垒
- 全球CBAM蔓延：碳足迹成为新的市场准入门槛

### 7.2 投资机会矩阵

| 方向 | 确定性 | 代表公司/赛道 |
|------|--------|------------|
| 欧洲储能市场（阳光电源等）| 高 | 欧洲大储加速，阳光电源市场份额21% |
| 中东/东南亚储能大单 | 高 | 比亚迪15.1GWh沙特、菲律宾1.4GW+3.3GWh |
| 钠离子电池技术 | 中高 | 供应链安全冗余，新一代替代路线 |
| 本地化运营服务商 | 中 | 跨境合规SaaS、碳足迹核算服务 |
| 一带一路新能源基建 | 中 | 光伏+储能+EPC总包 |
| 欧洲建厂（电芯企业）| 中 | 宁德时代匈牙利/西班牙、隆基德国 |

### 7.3 需要警惕的风险

1. **美国关税进一步升级**（OBBBA执行力度超预期）
2. **欧洲CBAM扩展至光伏下游**
3. **原产地规则收紧**（USMCA 2026年7月审查）
4. **供应链地缘政治中断**（南海/太平洋海运通道风险）
5. **全球光伏结构性产能过剩**（价格战压垮利润）
