# 33 — 中国低空经济2026：eVTOL与万亿新赛道爆发元年

**研究日期：** 2026年3月29日  
**课题方向：** 新兴产业 · 低空经济 · 创业机会

---

## 一、核心结论

2026年是中国低空经济从"政策元年"迈向"场景验证元年"的关键转折点。市场规模预计达**1.5万亿元**（民航局预测），eVTOL商业化正式起步，无人机数量突破328万架。但**供给侧热、需求侧冷**的结构性矛盾突出——大量项目停留在演示性阶段，缺乏高频刚需支撑。**H1洞察：真正的机会在工业级无人机（农业植保、电力巡检、海上油气）和低空基础设施（起降场、充电网络、通信感知），而非载人eVTOL（商业化仍需5-10年）。**

---

## 二、市场规模与政策里程碑

### 2.1 市场规模预测

| 机构 | 预测 |
|------|------|
| 中国民航局 | 2025年1.5万亿元，2035年3.5万亿元 |
| 赛迪顾问 | 2023年5059.5亿元，2024年6702.5亿元（+32.5%） |
| eVTOL专项 | 2026年95亿元 |
| 低空物流 | 2025年1200-1500亿元，2035年4500-6050亿元 |

**关键数据：**
- 2025年实名登记无人机总数突破**328万架**（较2020年51.7万架增长534.4%）
- 累计飞行时长4530万小时，较2022年翻倍增长
- 持有无人机运营合格证企业超**1.4万家**，持证操控员超**22.5万人**
- 2025年上半年投融资事件**143起**，同比增长120%，其中82.3%流向整机制造企业

### 2.2 政策发展脉络

```
2021年2月  "低空经济"首次写入国家规划（中共中央、国务院《国家综合立体交通网规划纲要》）
2023年12月 中央经济工作会议：打造低空经济为战略性新兴产业
2024年7月  政府工作报告：列为"新增长引擎"；国家发改委成立低空经济发展司
2025年11月 国务院《关于加快场景培育和开放推动新场景大规模应用的实施意见》
2026年7月  新修订《民用航空法》施行：首次增设"发展促进"专章，明确300米以下低空分类分级管理规则
```

**法律框架：** 新版民用航空法（2026年7月生效）为eVTOL等新业态建立精准监管体系，首次明确300米以下低空的分类管理规则，但空域所有权归属问题仍待专项立法解决。

## 三、产业链四大板块

低空经济涵盖**低空制造、低空飞行、低空保障、低空综合服务**四大板块。

### 3.1 低空制造：中国优势最突出的环节

**"无人机领跑，eVTOL紧追"**——

- 民用无人机企业登记注册**1081家**，备案产品**3623种**
- 在研吨级无人机**近30款**，eVTOL**近70种**
- 累计70余款国产传统通航装备完成适航取证，18款民用无人机获适航批准

**全链条配套成熟，中国全球领先：**
- 无人机研发设计企业达**7.7万家**（企查查数据）
- 大疆占据全球消费级无人机主要市场份额
- 电池（欣旺达航空电池）、机身材料（中复神鹰）、飞控系统（纵横股份）等细分领域具备全球竞争力

**"新能源汽车产业链"是中国低空制造的核心底气：**
电池、电驱、电控"三电系统"技术溢出，直接赋能eVTOL和无人机制造业。

### 3.2 eVTOL：商业化起步，但大规模落地尚远

**全球对比：** 中国亿航（EHang）和AutoFlight已获得型号合格证（TC），领先于大多数西方竞争者（Joby、Archer等仍处于FAA认证阶段）。

**亿航案例：**
- 已在全球占有一定市场份额，西班牙国家警察局、沙特主权基金均已采购
- 售价超40万美元的载人机型已交付**超300架**
- 战略目标：开拓拉美市场（中东已与美国二分天下，欧洲锁定市场地位）

**贺天星（亿航副总裁）原话：**
> *"后面会越来越多，增长会越来越快……在中东地区我们已经与美国两分天下，欧洲我们也已经锁定市场地位，接下来我们想推动拉美市场。"*

**eVTOL的核心挑战：**
1. **适航认证周期长**：从型号合格到商业运营还需运营合格证每架次审批
2. **基础设施空白**：全国通用机场仅**487个**（截至2025年底），与目标差距巨大
3. **成本过高**：城市空中出行商业化需每架次成本大幅下降
4. **监管真空**：空域管理"军-民-商"三轨并行，协调机制缺失

### 3.3 低空运营服务：最具确定性的增长极

**低空物流（最强赛道）：**
- 大湾区首条跨海商业化航线（空中深中通道）：深圳至中山4小时配送
- 天津首条常态化无人机运输血液制品航线：14.6公里仅用18分钟
- 2吨级eVTOL贵州高原试飞：120公里39分钟

**农业植保（最成熟场景）：**
- 农业无人机保有量超**30万架**，服务耕地面积**4.6亿亩**
- 多光谱无人机巡田+处方图实现精准作业
- 效率对比：150亩果园人工喷药需4人3天，无人机1天完成

**电力巡检（高价值场景）：**
- 电力线路无人机巡检里程突破**400万公里**
- 激光雷达自动生成三维模型，精度超目视
- 夜间巡检成为可能，1天=传统班组1周工作量

**海上油气（新增量）：**
- 北部湾油田41座海上平台+2个陆地终端厂全面推广无人机作业
- TD550重载无人直升机：抗8级风，单次运输超200公斤物资
- 作业效率提升30%以上，全年节约船舶租赁和燃油费用近**1500万元**

## 五、基础设施：最大瓶颈，也是最大机会

### 5.1 现状

- 全国通用机场仅**487个**（截至2025年底），远低于需求
- 低空通信感知网络尚在验证试点阶段
- 5G-A网络已在全国330余城市启动部署，为低空飞行提供通信覆盖

### 5.2 三大基础设施方向

**1. 起降场/充换电网络**
深圳目标2026年建成1200个起降点，是当前存量的数倍。充换电基础设施（类似新能源汽车充电桩逻辑）是**最先受益的赛道**。

**2. 低空通信感知网络**
6G"空天地"一体化网络是低空经济的"神经中枢"，基站通感融合是"神经末梢"。5G-A的通感一体技术已成熟，可同时满足通信和低空飞行器感知定位需求。

**3. 低空数字化管控平台**
广东省已建成低空飞行综合管理服务平台，接入全要素低空数据资源。全国统一标准的低空管控平台是解决"看不见、叫不到、管不住"的关键。

## 六、结构性问题：为何"听起来很热，落地很难"？

### 6.1 Jamestown等机构的核心质疑

> *"低空经济的市场规模被显著夸大了。当前低空经济发展几乎完全由供给侧政策驱动，而非真实需求侧拉动。"*

**"用不起、没人用、难持续"**——许多项目停留在演示性、临时性阶段，缺乏高频刚需。

**案例警示：** 某城市宣称可在院坝用无人机送咖啡，但实际运送一杯咖啡的成本高达**四位数**——地面需要大量技术支持，甚至需要车辆持续跟踪。

### 6.2 三大核心瓶颈

| 瓶颈 | 说明 |
|------|------|
| 空域管理复杂 | 军-民-商三轨分离，多层审批链，缺乏联合决策机制 |
| 基础设施空白 | 起降点/充电桩/通信感知网络严重不足 |
| 规模化商业路径不清 | 大多数应用场景成本高于传统方案 |

## 八、关键风险

1. **政策落地不及预期**：空域管理改革进展缓慢，适航审批周期长
2. **需求侧迟迟不启动**：大量政府示范项目无法转化为真实商业需求
3. **成本居高不下**：eVTOL和大型无人机成本远高于传统方案
4. **安全事件黑天鹅**：大规模商业化前的安全事故可能引发监管收紧
5. **地方债务风险**：多地推出千亿级产业目标，实际落地能力存疑（参考雄安低空经济示范区）

---

## 九、总结

**2026年是中国低空经济的"场景验证元年"**，而非"载人飞行元年"。市场空间真实存在（1.5万亿），政策红利真实释放，法律框架正在完善。但**"供给热、需求冷"**的结构性矛盾意味着大多数项目无法实现商业自洽。

**真正的机会在于**：
- 已经跑通商业模式的**农业植保、电力巡检、海上油气**等领域
- **基础设施**（起降场+充换电+管控平台）作为政府基建投资受益方向
- 新能源汽车产业链向低空制造的技术溢出（电池、电机、电控）

**需要警惕的**：eVTOL载人飞行器估值偏高，商业化路径漫长；地方政府的千亿目标多有注水成分；大量消费级无人机制造商处于亏损竞争状态。
